# üéµ CalmPlayer

**CalmPlayer** is a high-performance, Material 3-based music player designed for modern Android devices. It focuses on fluid navigation, efficient resource management, and a clean user experience.

---

## üöÄ Key Features

* **Smart Folder Picker:** Integrated intro flow using the Storage Access Framework (SAF) to select and manage your music library seamlessly.
* **Material 3 Expressive UI:** Features the latest M3 design components, including expressive loading indicators and dynamic color schemes.
* **Optimized Performance:** Specifically tuned for devices with 4GB RAM (like the Oppo Reno 3) by removing heavy GPU-bound blur effects and optimizing memory footprint.
* **Bouncy Gestures:** Intuitive MiniPlayer controls with smooth, bouncy swipe-up to expand and swipe-down to minimize animations.
* **Automated CI/CD:** Fully automated build and release pipeline using GitHub Actions with Semantic Versioning (`-alpha` track).
